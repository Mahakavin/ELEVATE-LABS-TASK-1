# Java Calculator

A simple command-line calculator built in Java.

## Features
- Addition, Subtraction, Multiplication, Division
- Looping option to continue calculations
- Input handling with Scanner

## Concepts Implemented
- **Packages** (`calc` package for organization)
- **Classes & Objects** (Calculator class, object creation)
- **Methods** (add, subtract, multiple, divide)
- **Control Flow** (if-else, while loop)
- **User Input** (Scanner for reading numbers and choices)

## How to Run
Compile with `javac calc/Intern_1_Calculator.java`  
Run with `java calc.Intern_1_Calculator`
